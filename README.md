tictactoe
=========

Smart tic tac toe players that learns from previous games.


Explanation
--------


games are positions on screen
123
456
789

example game:

g = "51238"

equals: first player wins
        x o x
        _ o _
        _ o _

        5 1 2 3 8 
        o x o x o

**player 1:** 0

**player 2:** x

games of odd length, are won by player1, o
games of even length, are won by player2, x
